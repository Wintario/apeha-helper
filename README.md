# Apeha Helper

Короткая инструкция для пользователя.

## Установка
1. Откройте `chrome://extensions`.
2. Включите "Режим разработчика".
3. Нажмите "Загрузить распакованное расширение".
4. Выберите папку `extension`.

## Как пользоваться
- Откройте страницу Apeha. Кнопка `Helper` появляется автоматически.
- Перетащите кнопку в удобное место (у правого края она "прилипает").
- Нажмите кнопку, чтобы раскрыть/свернуть окно helper.
- Во вкладке `Игра` находятся игровые переключатели и статусы.

## Быстрые действия
- `Ctrl+ЛКМ` по нику в списках живых/мертвых: добавить в отслеживание.
- `Ctrl+ЛКМ` по нику в правой панели статусов: добавить в отслеживание.
- `Ctrl+ЛКМ` по иконке игрока на карте (включая клонов): добавить в отслеживание.
- `Ctrl+ПКМ` по кнопке `Helper`: полностью скрыть helper.
- `Ctrl+ПКМ` по часам в `d_menu`: снова показать helper.
- В отдельном окне боя (`animate_bid...`): `Ctrl+ПКМ` по пустому месту, чтобы вернуть helper.

## Что делает helper
- Ведет watch-листы игроков (несколько блоков).
- Показывает статус строки: жив / мертв / неизвестно.
- Подсвечивает отслеживаемых на карте.
- Во вкладке `Игра`:
  - подсвечивает вашу заявку в комнате заявок,
  - отслеживает начало боя по поданной заявке и подает звуковой сигнал,
  - управляет звуком сигналов,
  - умеет следить за иммунитетом к нападению по странице персонажа.
- Показывает метки эффектов на карте:
  - синий щит (неуязвим),
  - черный щит (иммунитет к боевой магии),
  - `П` (прокля),
  - `К` (клич),
  - смайлик (сведен с ума),
  - снежинка (заморожен),
  - розовая маска (испугался).
- Рядом с картой показывает панель статусов:
  - `прокля`, `клич`, `сведен с ума`, `заморожен`, `испугался`, `неуязвим`, `иммунитет`.
  - панель можно перетаскивать и закрывать крестиком.
- Учитывает переключатель команды:
  - выбран синий -> показываются синие,
  - выбран красный -> показываются красные.
- Парсит только текущий раунд лога боя (старые раунды не влияют на подсветки текущего).
- Слежение за иммунитетом:
  - берет ссылку на персонажа из `d_pers`,
  - автоматически проверяет иммунитет примерно раз в 10-15 минут,
  - показывает оставшееся время в меню `Игра`,
  - кнопка `↻` рядом с чекбоксом запускает немедленную проверку,
  - при остатке меньше 15 минут подает звуковой сигнал и показывает розовое предупреждение слева от кнопки `Helper`.
- Подсветка заявки и старт боя:
  - выделяет вашу заявку в списке,
  - если пользователь неактивен, может обновлять комнату заявок, чтобы не пропустить старт,
  - после начала боя подает звуковой сигнал.

## Сохранение состояния
- Положение, раскрытие/сворачивание и hidden-режим сохраняются после обновления страницы.
- Настройки вкладки `Игра` и последнее состояние отслеживания иммунитета тоже сохраняются.
