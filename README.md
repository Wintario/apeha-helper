# Apeha Helper

Расширение-помощник для боев Apeha: отслеживание статусов, подсветки на карте и удобный watch-list игроков.

## Установка (кратко)
1. Скачайте репозиторий.
2. Откройте в Chrome/Chromium: `chrome://extensions`.
3. Включите "Режим разработчика".
4. Нажмите "Загрузить распакованное расширение".
5. Выберите папку `extension`.

## Что умеет расширение
- Кнопка `Helper` показывается всегда при наличии боевого контекста (в т.ч. арена/отдельное окно боя).
- Если бой открыт отдельной страницей (`animate_bid...`), вернуть helper можно `Ctrl+ПКМ` по свободной области.
- `Ctrl+ПКМ` по кнопке Helper скрывает helper (`hidden`), `Ctrl+ПКМ` по часам (`#clock` во фрейме `d_menu`) показывает обратно.
- В `hidden`-режиме отключаются все элементы helper: панели, бейджи, подсветки и парсинг эффектов.
- Состояния сохраняются между обновлениями: позиция кнопки/окна, раскрыт/свернут, hidden.
- Перетаскивание кнопки: есть "магнит" к правой границе (не выезжает за экран).

- Watch-блоки игроков:
- несколько блоков отслеживания;
- выбор активного блока;
- добавление/удаление строк;
- исправлена ширина после добавления/удаления блоков (без сжатия первичного блока).

- Переключатель команд инвертирован по смыслу:
- выбран синий -> показываются синие;
- выбран красный -> показываются красные.

- Быстрое добавление в отслеживание (`Ctrl+ЛКМ`):
- по нику в `aliveshow/deadshow`;
- по нику в правой панели статусов;
- по иконке игрока на карте (`pr_*`), включая клонов.

- Подсветки и метки на карте:
- обычная подсветка отслеживаемых;
- красная подсветка "поднят в прокле" (до конца текущего раунда);
- синий щит (слева в ячейке);
- черный щит (справа в ячейке, иммунитет к боевой магии, sticky до конца боя);
- яркая снежинка для заморозки;
- яркий "сумасшедший" смайлик (и в helper, и снизу иконки на карте до конца текущего раунда).

- Правая панель статусов рядом с картой:
- секции: `прокля`, `клич`, `сведен с ума`, `заморожен`, `неуязвим`, `иммунитет`, `поднят в прокле`;
- фильтрация по выбранной команде;
- ники кликабельны (`/info.html?nick=...`);
- `Ctrl+ЛКМ` по нику добавляет в watch-list;
- панель можно перетаскивать и закрывать крестиком;
- при входе в новый бой панель появляется снова.

- Парсинг логов по текущему раунду:
- учитывается только текущий раунд;
- корректная обработка маркера `Раунд N`/`Раунд № N` (в т.ч. с таймстампом в начале строки);
- логика "Оживить соратника": строка `вмешался в бой` + соседняя строка ниже с `использовал свиток ... Оживить соратника`;
- красная revive-подсветка применяется только к `pb0d/pb1d` (`png/gif`) и сбрасывается при смене раунда.

- Совместимость с фреймовой ареной и standalone-боем:
- правая панель рендерится через отдельный overlay-слой;
- верстка карты/позиции юнитов не ломается.
